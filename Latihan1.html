HTML atau Hyper Text Markup Langua adalah bahasa markah yang digunakan untuk membuat sebuah halaman web dan memberitahu browser bagaimana menampilkan konten berupa teks, gambar, tabel, form, hyperlink, dll.

CSS adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dokumen. Ada dua sifat css yaitu internal dan eksternal. CSS internal dimasukkan secara langsung ke dokumen web yang akan didesain.
